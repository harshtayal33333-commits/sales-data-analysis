# E-Commerce Sales Analysis
This project analyzes the UCI Online Retail dataset to understand sales performance and customer purchasing behavior.
## About the Dataset
The dataset contains transactional data of a UK-based online retail store with over 500,000 records.

## What I Did
- Loaded and cleaned the dataset using Pandas
- Removed missing values and negative quantities
- Created a Revenue column (Quantity × UnitPrice)
- Calculated total revenue generated
- Performed monthly sales trend analysis
- Identified top-selling products
- Analyzed country-wise revenue contribution

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Result
- Total Revenue: 8.3M+
- United Kingdom contributed the highest sales
- Clear monthly sales trend observed

## How to Run
Open the notebook in Google Colab, upload the dataset file, and run all cells.
